## 🔧 How to Update Your Website

### Update Frontend:
1. Make changes to code locally
2. Test: `npm run dev`
3. Commit: `git add . && git commit -m "Your message"`
4. Push: `git push origin main`
5. Vercel auto-deploys in 2-3 minutes!

### Update Backend:
1. Make changes to backend code

## 🆘 Troubleshooting

### Website not loading:
1. Check DNS propagation: https://dnschecker.org
2. Clear browser cache
3. Try incognito mode
4. Check Vercel deployment status

### Contact form not working:
1. Check browser console (F12)
2. Verify API URL in environment variables
3. Check backend logs in Render
4. Test API directly with curl

### Email not sending:
1. Verify Gmail app password is correct
2. Check backend logs for errors
3. Test email configuration
4. Ensure 2-step verification is enabled

### Backend sleeping (free tier):
1. First request takes 50 seconds (normal)
2. Subsequent requests are instant
3. Upgrade to paid tier for always-on
4. Or use a ping service to keep it awake
